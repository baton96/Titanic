# INTRO
This projects was created for the course of "Machine Learning" on the International Hellenic University in Thessaloniki during my Erasmus stay in the spring/summer semester 2019.
Project consistis of a model that predicts survival of Titanic's passengers in Kaggle's *"legendary Titanic ML competition"* as it was called by the creators of this competition.
More information about this competition can be found on it's [official site](https://www.kaggle.com/c/titanic).
# DESCRIPTION
The competition is simple: use machine learning to create a model that predicts which passengers survived the Titanic shipwreck.
